# ⚠️ Mixed-Questions

🔥 A lightweight repository packed with **mixed questions** for practice and revision.  
⚡ Designed to help learners cover the **remaining basics** across different topics — fast, sharp, and effective.

---

## 📘 Overview
- ⚡ Collection of short, simple, but challenging questions.  
- 💥 Covers multiple categories (Math, Logic, General Knowledge, etc.).  
- 🚨 Perfect for self‑practice, group quizzes, or quick revision under pressure.  

---

## 📂 Structure
- `questions/` → Sets of mixed questions.  
- `answers/` → Solutions or hints.  
- `README.md` → Project overview and usage guide.  

---

## 🚀 Goals
- Strengthen fundamentals through quick practice.  
- Provide modular sets of questions for different topics.  
- Encourage learners to test themselves under time pressure.  

---

This repository is intended for **learning and practice purposes**.  
Suggestions, improvements, and contributions are always welcome!
