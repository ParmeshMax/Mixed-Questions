# рқҗҢрқ—Ӯрқ—‘рқ–ҫб‘Ҝ-рқҗҗП…рқ–ҫрқ—Ңрқ—Қрқ—Ӯвұәрҗ“Ј

рқҗҖ бҘЈрқ—Ӯрқ—ҖЙҰрқ—ҚПүрқ–ҫрқ—Ӯрқ—ҖЙҰрқ—Қ рқ—Ӣрқ–ҫПҒвұәрқ—Ңрқ—Ӯрқ—Қвұәрқ—Ӣрқ—’ рқ–јвұәрҗ“Јрқ—ҚОұрқ—Ӯрҗ“Јрқ—Ӯрҗ“Јрқ—Җ рқ—Ңрқ—Ӯкӯ‘ПҒбҘЈрқ–ҫ кӯ‘рқ—Ӯрқ—‘рқ–ҫб‘Ҝ рқ—ҠП…рқ–ҫрқ—Ңрқ—Қрқ—Ӯвұәрҗ“Јрқ—Ң рқ–ҝвұәрқ—Ӣ ПҒрқ—ӢОұрқ–јрқ—Қрқ—Ӯрқ–јрқ–ҫ Оұрҗ“Јб‘Ҝ рқ—Ӣрқ–ҫрқ—Ҹрқ—Ӯрқ—Ңрқ—Ӯвұәрҗ“Ј.  
рқҗ“ЙҰрқ—Ӯрқ—Ң ПҒрқ—Ӣвұәрқ—ғрқ–ҫрқ–јрқ—Қ рқ—Ӯрқ—Ң б‘Ҝрқ–ҫрқ—Ңрқ—Ӯрқ—Җрҗ“Јрқ–ҫб‘Ҝ рқ—Қвұә ЙҰрқ–ҫбҘЈПҒ бҘЈрқ–ҫОұрқ—Ӣрҗ“Јрқ–ҫрқ—Ӣрқ—Ң рқ–јвұәрқ—Ҹрқ–ҫрқ—Ӣ рқ—ҚЙҰрқ–ҫ **рқ—Ӣрқ–ҫкӯ‘Оұрқ—Ӯрҗ“Јрқ—Ӯрҗ“Јрқ—Җ б‘ІОұрқ—Ңрқ—Ӯрқ–јрқ—Ң** Оұрқ–јрқ—Ӣвұәрқ—Ңрқ—Ң б‘Ҝрқ—Ӯрқ–ҝрқ–ҝрқ–ҫрқ—Ӣрқ–ҫрҗ“Јрқ—Қ рқ—ҚвұәПҒрқ—Ӯрқ–јрқ—Ң.

---

## рҹ“ҳ рқҗҺрқ—Ҹрқ–ҫрқ—Ӣрқ—Ҹрқ—Ӯрқ–ҫПү
- рқҗӮвұәбҘЈбҘЈрқ–ҫрқ–јрқ—Қрқ—Ӯвұәрҗ“Ј вұәрқ–ҝ рқ—ҢЙҰвұәрқ—Ӣрқ—Қ, рқ—Ңрқ—Ӯкӯ‘ПҒбҘЈрқ–ҫ рқ—ҠП…рқ–ҫрқ—Ңрқ—Қрқ—Ӯвұәрҗ“Јрқ—Ң.
- рқҗӮвұәрқ—Ҹрқ–ҫрқ—Ӣрқ—Ң кӯ‘П…бҘЈрқ—Қрқ—ӮПҒбҘЈрқ–ҫ рқ–јОұрқ—Қрқ–ҫрқ—Җвұәрқ—Ӣрқ—Ӯрқ–ҫрқ—Ң (кӯ‘Оұрқ—ҚЙҰ, бҘЈвұәрқ—Җрқ—Ӯрқ–ј, рқ—Җрқ–ҫрҗ“Јрқ–ҫрқ—ӢОұбҘЈ рқ—„рҗ“ЈвұәПүбҘЈрқ–ҫб‘Ҝрқ—Җрқ–ҫ, рқ–ҫрқ—Қрқ–ј.).
- рқҗ”рқ—Ңрқ–ҫрқ–ҝП…бҘЈ рқ–ҝвұәрқ—Ӣ рқ—Ңрқ–ҫбҘЈрқ–ҝ-ПҒрқ—ӢОұрқ–јрқ—Қрқ—Ӯрқ–јрқ–ҫ, рқ—Җрқ—ӢвұәП…ПҒ рқ—ҠП…рқ—ӮЖ¶Ж¶рқ–ҫрқ—Ң, вұәрқ—Ӣ рқ—ҠП…рқ—Ӯрқ–јрқ—„ рқ—Ӣрқ–ҫрқ—Ҹрқ—Ӯрқ—Ңрқ—Ӯвұәрҗ“Ј.

## рҹ“Ӯ рқҗ’рқ—Қрқ—ӢП…рқ–јрқ—ҚП…рқ—Ӣрқ–ҫ
- `рқ—ҠП…рқ–ҫрқ—Ңрқ—Қрқ—Ӯвұәрҗ“Јрқ—Ң/` вҶ’ рқҗ’рқ–ҫрқ—Қрқ—Ң вұәрқ–ҝ кӯ‘рқ—Ӯрқ—‘рқ–ҫб‘Ҝ рқ—ҠП…рқ–ҫрқ—Ңрқ—Қрқ—Ӯвұәрҗ“Јрқ—Ң.
- `Оұрҗ“Јрқ—ҢПүрқ–ҫрқ—Ӣрқ—Ң/` вҶ’ рқҗ’вұәбҘЈП…рқ—Қрқ—Ӯвұәрҗ“Јрқ—Ң вұәрқ—Ӣ ЙҰрқ—Ӯрҗ“Јрқ—Қрқ—Ң.
- `рқҗ‘рқҗ„рқҗҖрқҗғрқҗҢрқҗ„.кӯ‘б‘Ҝ` вҶ’ рқҗҸрқ—Ӣвұәрқ—ғрқ–ҫрқ–јрқ—Қ вұәрқ—Ҹрқ–ҫрқ—Ӣрқ—Ҹрқ—Ӯрқ–ҫПү.
